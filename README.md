🌱 Eco Smart - Sistema Inteligente de Detecção de Gases










📖 Sobre o Projeto

O Eco Smart é um projeto desenvolvido com o objetivo de promover a segurança em ambientes residenciais e empresariais por meio da detecção de gases potencialmente perigosos.

O sistema utiliza um sensor conectado a um Arduino Uno para monitorar continuamente o ambiente, emitindo alertas visuais e sonoros sempre que níveis de risco são identificados.

Além do protótipo físico, foi desenvolvido um site interativo para apresentar o funcionamento do projeto, seus benefícios, componentes utilizados e demonstrações visuais do sistema em ação.
