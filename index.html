Componentes do Eco Smart

O projeto utiliza Arduino e componentes eletrônicos para detectar gases no ambiente e emitir alertas visuais e sonoros quando houver risco.

⚡

Arduino Uno

Responsável por receber os dados do sensor e controlar os alertas do sistema.

🌫️

Sensor MQ-2

Detecta gases inflamáveis e fumaça, enviando valores analógicos para o Arduino.

🚨

Buzzer

Emite um alerta sonoro quando o nível de gás ultrapassa o limite definido.

💡

LEDs Indicadores

Mostram visualmente o estado do ambiente: seguro, atenção ou perigo.

Protótipo e Código

Implementação do Eco Smart

🖼️ Protótipo do Circuito
💻 Código Arduino — Eco Smart

// Código Arduino Eco Smart

int sensor = A0;
int ledVerde = 7;
int ledVermelho = 8;
int buzzer = 9;

void setup() {
  pinMode(ledVerde, OUTPUT);
  pinMode(ledVermelho, OUTPUT);
  pinMode(buzzer, OUTPUT);

  Serial.begin(9600);
}

void loop() {
  int valor = analogRead(sensor);

  Serial.println(valor);

  if (valor < 400) {
    digitalWrite(ledVerde, HIGH);
    digitalWrite(ledVermelho, LOW);
    noTone(buzzer);
  } else {
    digitalWrite(ledVerde, LOW);
    digitalWrite(ledVermelho, HIGH);
    tone(buzzer, 1000);
  }

  delay(500);
}
Como Funciona

Processo em 4 etapas

Entenda o fluxo completo do sistema Eco Smart, da coleta de dados até o alerta de segurança.

1
🌫️

Detecção

O sensor MQ-2 capta as concentrações de gases presentes no ar, como CO, metano e fumaça, com alta sensibilidade.

2
🔌

Processamento

O Arduino Uno lê os sinais analógicos do sensor, converte em valores PPM e processa os dados em tempo real.

3
📊

Análise

O sistema compara os valores com limites de segurança pré-definidos e classifica o ambiente como seguro, alerta ou perigo.

4
🚨

Alerta

Em caso de risco, o buzzer soa, o LED vermelho acende e o display exibe o tipo de gás detectado e o nível de risco.
